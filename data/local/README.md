# Local governed data

Place the human-reviewed B&NES ATM reference at
`data/local/banes-atm-full.geojson` to run ATM comparison. The repository does not
redistribute that geometry by default. The reference configuration keeps comparison
disabled until the operator has supplied the file and deliberately enabled it.

The current reference corpus was retrieved from the public B&NES Active Travel
Masterplan portal WFS. Public accessibility alone is not treated as permission to
republish the geometry.
